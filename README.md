# TA for ITREX 


## Installation
Clone from GIT:
```sh
$ git clone <this repo>
```

Install dependencies:
```sh
npm i
```

run ongod:
```sh
$ mongod
```

Run seeder:
```sh
$ node server/db/seed
```

Run backend:
```sh
$ npm run backdev
```

Run front:
```sh
$ npm run ngdev
```
